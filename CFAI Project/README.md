# Hospital Bed Allocation System

## Overview

The Hospital Bed Allocation System is an AI-based healthcare management project developed to automate and optimize hospital bed allocation. The system helps hospitals efficiently assign beds to patients based on priority, severity, and resource availability.

This project combines concepts from Artificial Intelligence such as search algorithms, constraint satisfaction, heuristic-based decision-making, and priority scheduling to improve healthcare resource management.

---

# Features

* Patient registration system
* Bed availability management
* Priority-based patient allocation
* ICU, General, and Emergency bed handling
* Constraint-based allocation
* Waiting list management
* Allocation logging and tracking
* Explainable allocation reports
* Emergency patient prioritization

# System Workflow

1. Register patient details
2. Assess patient severity
3. Check bed availability
4. Apply allocation constraints
5. Select optimal bed
6. Allocate bed
7. Update hospital records
8. Generate allocation report
